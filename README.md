# This is a simple and efficient Python script designed to automate the process of splitting household expenses among roommates.
# This tool takes the guesswork out of monthly calculations by consolidating rent, food, and utility costs into a single, easy-to-read breakdown.The program is written in Python and uses basic arithmetic logic to ensure a fair and accurate distribution of payments for everyone in a shared living space.

# Features
# Custom Occupancy: Calculates shares based on any number of persons in a single room.
# Integrated Utility Billing: Specifically calculates electricity costs by multiplying units spent by the charge per unit.
# Food Expense Tracking: Includes a dedicated input for shared snacking or food orders.
# Instant Calculation: Provides both the grand total for the room and the individual amount due per person.
# Calculation Logic
# Step 1: Calculate total electricity cost by multiplying units by charge per unit.
# Step 2: Sum the Rent, Food, and Electricity cost for the Total Amount.
# Step 3: Divide the Total Amount by the number of persons for the Individual Share.
# How to Run
# 1. Ensure you have Python installed on your system.
# 2. Download or clone the rent_calculator.py file.
# 3. Run the script using the following command in your terminal:
# python rent_calculator.py
# 4. Follow the on-screen prompts to enter your monthly data.
